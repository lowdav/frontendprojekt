Följ internationella rymdstationen ISS i realtid

  • ISS koordinater och information om hastighet, höjd och synlighet hämtas från ett API.
  • Om den första datakällan inte kan leverera svar hämtas koordinater från ett sekundärt API som endast har koordinater.
  • Koordinaterna används för att hämta en Google-karta över platsen och placera ut en ikon föreställande ISS vid aktuell position på kartan.
  • Koordinaterna används igen för så kallad reverse geokodning, som hämtar informaton om vilken plats koordinaterna tillhör. Även detta från Google, men från ett annat API.
  • Ett tredje API hämtar aktuell temperatur för platsen.
